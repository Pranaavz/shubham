** Weather Forecast App **

A responsive weather forecast web application built with **React**, **TypeScript**, and **Vite**.

## Features

* Search weather by city name
* Real-time weather updates
* Hourly forecast
* Daily forecast
* Favorite cities support
* Clean and responsive UI

## Tech Stack

* React
* TypeScript
* Vite
* CSS
* Open-Meteo API

## Installation

1. Clone the repository:


git clone https://github.com/pranaavz/weather-forecast-app.git


2. Install dependencies:

npm install


3. Run the project:


npm run dev
